## ✨ Features

| Feature | Description |
|---------|-------------|
| 🎯 **Match scoring** | Every job scored 1-10 against your background |
| ✈️ **Visa detection** | Automatically flags visa and relocation packages |
| 🔍 **Dual search** | Searches twice with different queries for better results |
| 📥 **Download** | Export ranked job list as text file |
| 🤖 **Groq LLaMA** | Free AI analysis — no paid API key needed |
| 🌍 **12 countries** | Region-specific job boards for each country |

---

## 🏗️ How it works
👤 You enter background + select region + job type
↓
🔍 DuckDuckGo search × 2 (different queries)
↓
📋 Search results collected
↓
🤖 Groq LLaMA reads results and extracts job details
↓
🎯 Each job scored 1-10 against your background
↓
✈️ Visa/relocation packages flagged
↓
📊 Results ranked highest → lowest match score
↓
💾 Download as text file

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/subrahmanyahegdeuk-eng/nz-job-scout.git
cd nz-job-scout

# Install dependencies
pip install -r requirements.txt

# Run the app
streamlit run app.py
```

> Get your **free** Groq API key at [console.groq.com](https://console.groq.com)

---

## 📋 Usage
Enter your Groq API key (free at console.groq.com)
Paste your background in 2-3 sentences
Select your target region and job type
Set number of results (3-6)
Click Search
Get ranked results in 15-30 seconds
Download results as text file
